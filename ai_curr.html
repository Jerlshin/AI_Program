PHASE 1: Foundational Competence & Data Toolkit

(4-6 Weeks) Establish a rock-solid, practical foundation in Python, data manipulation, and developer tools.

Progress 15%

1. Python Mastery

  • Core: Core Syntax, OOP, Data Structures, Basic Algorithms, Virtual Environments.
  • Mandatory: Deep dive into Python fundamentals.

Enhanced Content:

  • Python testing (pytest)
  • Debugging techniques
  • Code documentation (docstrings, type hints)
  • Basic software design patterns

2. Data Science Stack

  • Core: Arrays, Linear Algebra (NumPy), DataFrames (Pandas), Data Visualization (Matplotlib).
  • Elective (NumPy): Numpy
  • Elective (Pandas): Pandas
  • Elective (Matplotlib): Matplotlib

3. Development Environment

  • Core: Command Line (Bash), Version Control (Git), Linux basics, REST API concepts.
  • Secondary: Git, Linux, REST API, FastAPI basics.
  • Elective (Bash): Bash

4. Neural Networks Intro

  • Core: Conceptual understanding of what a neural network is and how it learns (before diving into Deep Learning frameworks).
  • Mandatory (NN): Neural Networks

Mathematics Fundamentals:

  • Linear Algebra (vectors, matrices, eigenvalues)
  • Calculus (differentiation, gradients)
  • Probability & Statistics (distributions, hypothesis testing)
  • Basic Optimization concepts

PHASE 2: Core Machine Learning & MLOps Basics

(6-8 Weeks) Master classical ML algorithms, the modeling pipeline, and essential MLOps concepts.

Progress 35%

1. Classical ML

  • Core: Supervised Learning (Regression, Classification), Unsupervised Learning (Clustering, Dimensionality Reduction).
  • Mandatory (Models): ML models (Video)
  • Mandatory (Models): ML Models (Article)

Additional Algorithms:

  • Ensemble methods (Random Forest, Gradient Boosting - XGBoost, LightGBM)
  • Time Series Analysis and forecasting
  • Anomaly Detection techniques
  • Recommendation Systems (collaborative filtering, content-based)

2. Modeling Workflow

  • Core: Scikit-learn Pipelines, Feature Engineering, Model Training, Hyperparameter Tuning.
  • Additional: Scikit-learn, Metrics, Data Engineering.

3. Model Evaluation

  • Core: Metrics (Precision, Recall, F1, AUC, RÂ²), Confusion Matrices, Bias-Variance Tradeoff.
  • Focus: Focus on choosing the right metric for the business problem.

4. Experiment Tracking

  • Core: Introduction to tracking models, parameters, and metrics for reproducibility.
  • Additional: MLflow basics, Scikit-learn pipelines.
  • Elective (What is MLOps): What is MLOps
  • Elective (MLOps): MLOps

MLOps Foundations:

  • Data versioning (DVC)
  • Basic model serialization
  • Introduction to ML pipelines

PHASE 3: Deep Learning & Representation Architectures

(8-10 Weeks) Master Deep Learning fundamentals and specialized architectures for Vision and Sequence data.

Progress 55%

1. DL Frameworks

  • Core: Core use of TensorFlow and PyTorch for building and training simple neural networks.
  • Mandatory (TF Course): Tensorflow full course
  • Mandatory (PyTorch): PyTorch
  • Elective (PyTorch Course): PyTorch full course
  • Mandatory (Choice): Choice of libraries
  • Mandatory (Choosing): Choosing the right one

2. Advanced DL Concepts

  • Core: Optimizers (SGD, Adam), Regularization, Transfer Learning, Fine-Tuning.
  • Additional: Optimizers, Regularization, Transfer Learning, Fine-tuning.

3. Sequence Models

  • Core: Recurrent Neural Networks (RNNs), Long Short-Term Memory (LSTM), Attention Mechanism.
  • Elective (RNN): RNN
  • Elective (LSTM): LSTM
  • Mandatory (Attention): Attention in Transformers

4. CV & Optimization

  • Core: Convolutional Neural Networks (CNNs), Model Pruning, Quantization, Distillation (conceptual).
  • Additional: Quantization, Pruning, Distillation.

PHASE 3.5: Specialized ML Applications

(8-10 Weeks) Deep dive into specialized applications of machine learning across different domains.

Progress 65%

Computer Vision Deep Dive

  • Core: Object detection (YOLO, Faster R-CNN)
  • Core: Image segmentation (U-Net, Mask R-CNN)
  • Core: Image generation (VAEs, GANs basics)
  • Core: Optical Character Recognition (OCR)

Natural Language Processing

  • Core: Text preprocessing and feature extraction
  • Core: Word embeddings (Word2Vec, GloVe)
  • Core: Named Entity Recognition (NER)
  • Core: Sentiment analysis
  • Core: Text classification

Time Series & Forecasting

  • Core: ARIMA, Prophet
  • Core: LSTM for time series
  • Core: Feature engineering for temporal data

Industry Applications

  • Core: Healthcare AI (medical imaging, drug discovery)
  • Core: Financial AI (fraud detection, algorithmic trading)
  • Core: Retail AI (demand forecasting, personalization)
  • Core: Manufacturing AI (predictive maintenance, quality control)

PHASE 4: Generative AI, LLMs, and Agentic Systems

(8-10 Weeks) Master the modern AI landscape, from Transformers and RAG to building multi-step AI agents.

Progress 85%

1. Transformer Ecosystem

  • Core: Transformer Architecture (BERT, GPT), Prompt Engineering, Fine-tuning, Open-source model usage.
  • Additional: HuggingFace Transformers, Transformer Architecture.
  • Elective (Transfer): Transfer learning
  • Elective (Prompt Eng): Prompt Engineering
  • Mandatory (HF): HuggingFace
  • Elective (Quantize): Quantization Fundamentals

Advanced LLM Topics:

  • Fine-tuning techniques (LoRA, QLoRA, P-tuning)
  • Model quantization (GGUF, GPTQ, AWQ)
  • Prompt engineering advanced patterns (chain-of-thought, self-consistency)
  • LLM evaluation frameworks (HELM, MT-Bench)
  • Multimodal models (GPT-4V, LLaVA, CLIP)

2. RAG & Vector Databases

  • Core: RAG fundamentals, chunking, embedding, vector search, and choosing a Vector Database.
  • Additional: RAG, Vector Databases.
  • Elective (Vector DBs): Building Applications with Vector Databases
  • Elective (Retrieval): Advanced Retrieval for AI with Chroma

Advanced RAG:

  • Query transformation
  • Hybrid search (keyword + vector)
  • Reranking models
  • Multi-hop RAG
  • Graph RAG

3. LLM Orchestration & Agents

  • Core: LangChain/LlamaIndex for applications, LangGraph for multi-step agents, and evaluation of RAG systems.
  • Mandatory (LangChain): LangChain for LLM Application Development
  • Mandatory (Chat Data): LangChain: Chat with Your Data
  • Mandatory (LLMs Work): How Transformer LLMs Work
  • Mandatory (Eval RAG): Building and Evaluating Advanced RAG
  • Mandatory (Agentic RAG): Building Agentic RAG with LlamaIndex
  • Mandatory (LangGraph): AI Agents in LangGraph
  • Elective (Workflows): Event-Driven Agentic Document Workflows

AI Agents:

  • Tool usage (ReAct pattern)
  • Multi-agent systems
  • Planning and reasoning (Tree of Thought, Graph of Thought)
  • Agent evaluation (AgentBench, WebArena)

4. Advanced Applied CV

  • Core: Object Detection (YOLO), Semantic Segmentation (U-Net), OCR.
  • Mandatory (YOLO): YOLO
  • Mandatory (U-Net): U-Net
  • Elective (Multimodal): Building Multimodal Search and RAG

PHASE 5: MLOps, Cloud Deployment, & Job Readiness

(4-6 Weeks) Deploy, monitor, and maintain models in a professional setting and prepare for technical interviews.

Progress 100%

1. Production Deployment

  • Core: Containerization (Docker), REST APIs (FastAPI), Serverless functions, Model Serving.
  • Additional: FastAPI, Docker basics.

Specialized Deployment:

  • Edge deployment (TensorFlow Lite, ONNX Runtime)
  • Mobile ML (Core ML, ML Kit)
  • Real-time inference optimization
  • Batch inference patterns

2. Cloud Platforms

  • Core: Hands-on deployment practice on one major cloud provider's ML stack.
  • Additional: Choose one of SageMaker (AWS), Azure ML, or Google Cloud Vertex AI.

Cloud & Infrastructure:

  • Container orchestration (Kubernetes basics)
  • Infrastructure as Code (Terraform, CloudFormation)
  • ML platform design (Metaflow, Kubeflow)
  • Cost optimization for ML workloads

3. Pipeline Orchestration

  • Core: MLOps for traditional models and LLMs, including continuous integration/deployment (CI/CD).
  • Additional: MLOps basics.
  • Mandatory (LLMOps): LLMOps

Advanced MLOps:

  • Feature stores (Feast, Tecton)
  • Model registries
  • Data quality monitoring (Great Expectations)
  • Model monitoring (evidently.ai, Arize)
  • Drift detection (data drift, concept drift)

4. Job Readiness

  • Core: Data Structures & Algorithms (DSA), System Design (ML-focused), Resume/Portfolio creation.
  • Additional: Portfolio website, DSA, Interview Preparation. (Focus on designing ML systems like a Recommendation Engine or Real-time Fraud Detector).

Soft Skills & Business Acumen:

  • ML project management
  • Stakeholder communication
  • Business metrics translation
  • Experiment design and A/B testing
  • Cost-benefit analysis for AI projects

Advanced Computer Vision

  • Core: 3D computer vision
  • Core: Video understanding
  • Core: Medical imaging applications
  • Core: Autonomous vehicle perception

Emerging Technologies

  • Core: Federated Learning
  • Core: Differential Privacy
  • Core: Quantum Machine Learning (basic concepts)

Industry Best Practices

  • Core: ML design patterns
  • Core: Code quality for ML
  • Core: Reproducibility practices
  • Core: Team collaboration workflows
  • Core: Documentation standards
